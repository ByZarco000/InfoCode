Introducción a HTML

Ahora comencemos a ver los fundamentos de HTML. Aquí tenemos un ejemplo de una página web muy sencilla: 

                <!DOCTYPE html>
                <html lang="es">
                  <head>
                    <title>Mi página web</title>
                  </head>
                  <body>
                    <h1>Mi página web</h1>
                    <p>Estoy aprendiendo desarrollo web.</p>
                  </body>
                </html>

Tip: En HTML, la indentación (el espacio o sangría que colocamos antes de las líneas) es muy importante porque nos permite distinguir visualmente la estructura de la página web en el archivo. Aunque no es obligatoria, se recomienda usarla para comprender más fácilmente la estructura de la página. Se recomienda dejar 2 espacios por cada nivel de indentación.

Elementos

Un archivo HTML está conformado por elementos HTML. Estos elementos son componentes individuales en HTML que nos permiten crear la estructura de la página.

Este es un ejemplo de un elemento:

<h1>Mi página web</h1>

Tip: Ciertos elementos pueden contener a otros elementos y crear estructuras más complejas. Al anidar los elementos uno dentro del otro, se debería indentar los elementos internos para reflejar esta estructura, como en nuestro ejemplo anterior.

Etiquetas
                
    
                    
  • <html> - Todo el contenido de la página debe estar entre estas etiquetas.
    • 
                    
  • <head> - Este elemento contiene el título de la página que se muestra en la pestaña del navegador y metadatos adicionales.
    • 
                    
  • <body> - Aquí se encuentran todos los elementos visibles de la página web. La estructura principal de la página debe estar entre estas etiquetas.
    • 
                    
  • <h1>, <h2>, <h3>, <h4>, <h5>, <h6> - Estos elementos representan encabezados o títulos de diferente importancia en la página web.
    • 
                    
  • <p> - Se utiliza para crear párrafos de texto.
    • 
                    
  • <a> - Permite enlazar a otras páginas web o secciones internas de tu sitio.
    • 
                    
  • <strong> - Esta etiqueta resalta el texto importante y generalmente se muestra en negrita.
    • 
                    
  • <em> - Esta etiqueta resalta el texto importante y generalmente se muestra en cursiva.
    • 
                    
  • <form> - Te permite crear formularios interactivos.
    • 
                    
  • <hr> - Una línea horizontal utilizada para separar secciones de contenido en tu página web.
    • 
                    
  • <input> - Permite crear campos de entrada en formularios, como cuadros de texto y botones de radio.
    • 
                    
  • <footer> - El pie de página de tu página web.
    •

La mayoría de los elementos en HTML tienen una etiqueta de apertura y una etiqueta de cierre. Entre ambas etiquetas se escribe el contenido del elemento. En el ejemplo anterior estamos creando un elemento de tipo <h1> (un encabezado de nivel 1):</h1>

<h1>Mi página web</h1>
