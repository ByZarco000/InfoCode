HTML
HTML es el lenguaje de marcado utilizado para crear páginas web. Se compone de una serie de etiquetas que definen la estructura y el contenido de una página.
CSS
CSS es un lenguaje de estilo utilizado para controlar la presentación y el diseño de una página web.
JavaScript
JavaScript es un lenguaje de programación que se utiliza para crear interactividad en las páginas web.
